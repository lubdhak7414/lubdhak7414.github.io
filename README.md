# saf1.me

Personal portfolio, resume, and blog built with [Astro 6](https://astro.build/) + [Tailwind CSS 4](https://tailwindcss.com/). Deployed on [Cloudflare Pages](https://pages.cloudflare.com/).

Forked from [astro-resume](https://github.com/srleom/astro-resume) by srleom.

## Development

```bash
npm install
npm run dev       # dev server + type-check
npm run build     # astro check + build
npm run lint      # prettier + eslint
```

## Stack

- Astro 6 (static, Content Layer API)
- Tailwind CSS 4 (CSS-first config)
- ESLint 9 (flat config)
- MDX + Expressive Code
- Cloudflare Pages
